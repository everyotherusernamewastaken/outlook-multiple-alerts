# Icon Assets

You need to create the following icon files:

- `icon-16.png` - 16x16 pixels
- `icon-32.png` - 32x32 pixels  
- `icon-80.png` - 80x80 pixels

These icons will appear in Outlook's ribbon and add-in UI.

## Temporary Solution

For now, you can use any simple PNG images with the correct dimensions. You can create them using:
- Paint (Windows built-in)
- Online tools like Canva or Photopik
- Any image editor

Just save them with the exact filenames above in this folder.
